This is a simple FizzBuzz example. The game is to write out every number between 0 and some upper value but when you get to a specified multiple of 3 it prints out Fizz and when you get to a Multiple of 5 it prints out Buzz. And when the value is a multiple of both 3 and 5 it will print out FizzBuzz. This little example allows you to choose the relevent multiples as well as the upper limit of the set of numbers to check (within reason).
