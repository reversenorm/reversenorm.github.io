This is a simple FizzBuzz example. The game is to write out every number between 0 and some upper value but when the program gets to a specified multiple, for instance of 3, the program prints out Fizz. When it gets to a different multiple, for instance of 5, the program prints out Buzz. Finally when the value reachedis a multiple of both it will print out FizzBuzz. This little example allows you to choose the relevent multiples as well as the upper limit of the set of numbers to check (within reason).
